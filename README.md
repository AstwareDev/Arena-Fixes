## Installation

```bash
git clone https://github.com/AstwareDev/Arena-Fixes.git
```

1. Go to `chrome://extensions`
2. Enable **Developer mode** in the top-right corner
3. Click **Load unpacked**
4. Select the `Arena-Fixes` folder

Then head to [arena.ai](https://arena.ai). You're done.

## Privacy

No telemetry. No remote scripts. No hidden dependencies.

We ask for two permissions:
- `storage` — to remember your settings
- Access to `arena.ai` — to actually run

That's the whole list.
